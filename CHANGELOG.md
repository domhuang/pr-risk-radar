# Changelog

## v0.1.2 - 2026-06-13

- Updated PR comments in place instead of posting duplicate reports on every push.
- Added a stable hidden marker for PR Risk Radar comments.
- Kept support for legacy bot comments generated by earlier releases.

## v0.1.1 - 2026-06-11

- Fixed GitHub Action input handling for hyphenated input names.
- Passed the GitHub token explicitly in the example workflow.
- Made pull request comment failures non-blocking while still writing the Actions summary.

## v0.1.0 - 2026-06-11

- Added the first PR Risk Radar CLI.
- Added GitHub Action support for pull request summaries and comments.
- Added risk signals for auth, billing, migrations, infrastructure, dependencies, large diffs, missing tests, and missing docs.
- Added project documentation, CI, tests, issue templates, security policy, and MIT license.
